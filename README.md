# Zynq Sandbox

The purpose of this repository is for me to explore how I can apply my existing embedded knowledge to the Zync FPGA-SoCsand if the Zynq chip family is right for the Argus project.


## Glossary

Xilinx Support Archive(XSA): A *.xsa file is a Vivado generated handoff file that tells Vitis what hardware exists on the FPGA-SoC design for building a matching embedded software platform.
